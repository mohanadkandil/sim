# Simuul Product & User Data

## Product Overview

Simuul is an AI-powered social simulation platform that lets researchers and product teams model how communities of users respond to new features, content, or policy changes. Users upload research documents, and Simuul generates an interactive knowledge graph, simulated forum discussions, and behavioral dashboards.

## User Segments

### Researchers (312 users)
- Average 22 sessions per month
- Primary use: Modeling community reactions to policy changes
- Most used features: Graph exploration, forum simulation, activity dashboard
- Pain points: Slow graph generation for large documents, limited export options
- Satisfaction: 4.4/5

### Product Managers (541 users)
- Average 15 sessions per month
- Primary use: Pre-launch feature testing with simulated user feedback
- Most used features: Forum simulator, persona customization, reaction analysis
- Pain points: Simulated personas feel too uniform, want more demographic control
- Satisfaction: 4.1/5

### UX Designers (198 users)
- Average 9 sessions per month
- Primary use: Quick concept validation before user testing
- Most used features: Graph view, forum threads
- Pain points: Hard to distinguish between persona types in the graph
- Satisfaction: 3.9/5

### Churned Users (134 users)
- Left within first 60 days
- Top reasons for leaving:
  - Simulation results felt unrealistic (54%)
  - Missing integrations with existing tools (31%)
  - Pricing concerns (15%)
- Common feedback: "The personas all sounded the same"

### New Users (267 users)
- Joined in last 30 days
- Onboarding completion rate: 68%
- Most common first project: Customer feedback simulation
- Drop-off point: After generating the graph, unsure what to do next

## Key Features

### Knowledge Graph
- Extracts entities, concepts, and relationships from uploaded documents
- Interactive node-link visualization
- Color-coded by entity type
- User rating: 4.5/5

### Forum Simulator
- Generates realistic threaded discussions between simulated personas
- Personas are seeded from the uploaded document's user segments
- Supports multiple discussion topics per project
- User rating: 4.2/5

### Activity Dashboard
- Tracks simulation runs and engagement metrics
- Shows sentiment trends across persona segments
- Known issues: Dashboard data can lag by up to 30 seconds after simulation
- User rating: 3.8/5

### Persona Engine
- Automatically creates user personas from document analysis
- Assigns demographic traits, opinions, and communication styles
- Top request: Ability to manually edit or add personas
- User rating: 4.0/5
